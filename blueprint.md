# **App Name**: MentorAI

## Core Features:

- Student Profile Management: Create, read, update, and delete student profiles, including academic details, interests, and career goals.
- AI-Powered Profile Analysis: Analyze student data using Gemini AI to identify strengths, weaknesses, study suggestions, and career recommendations.
- Intelligent Chat Interface: Provide personalized mentorship through a chat interface, with context-aware responses and tailored advice using Google Gemini as a tool.
- Personalized Career Guidance: Generate detailed career roadmaps based on student profiles, including top career paths, skill development, and recommended courses using Gemini AI.
- AI Study Plan Generation: Create personalized weekly study schedules using Gemini AI, considering student data, grades, and focus subjects.
- Academic Progress Tracking: Track academic progress over time, compare new grades with previous grades, and re-analyze if needed.
- Firestore Integration: Store and manage student data, chat sessions, academic progress, and study plans using Firestore.

## Style Guidelines:

- Primary color: HSL(220, 70%, 50%) which converts to a medium blue (#3498DB), symbolizing trust, intelligence, and mentorship.
- Background color: A very light blue HSL(220, 20%, 95%) which converts to (#F0F8FF), offering a calm and clean backdrop.
- Accent color: HSL(190, 60%, 45%) which converts to a teal (#46B8B3), used for interactive elements and highlights, providing a sense of growth and progress.
- Body font: 'Inter', a grotesque-style sans-serif providing a modern and neutral feel for readability.
- Headline font: 'Space Grotesk', a proportional sans-serif lending a computerized, techy, scientific feel for headlines.
- Use consistent, clean icons from Lucide React or Heroicons to represent features and actions.
- Employ a card-based design with subtle shadows for depth and a clean, spacious layout.
